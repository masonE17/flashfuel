# FlashFuel
- Fuel your mind, flash your knowledge. 
- FlashFuel is a mobile flashcard app built with React Native and Expo. Create custom study sets, quiz yourself with multiple-choice questions, and track your progress — all backed by Supabase.

## Features
- **Auth** — sign up or log in with email/password
- **Create custom sets** — build flashcard sets with a subject, description, and multiple-choice questions (mark the correct answer for each)
- **Library** — browse all your sets, see card counts at a glance, delete sets you no longer need
- **Quiz mode** — answer one question at a time, submit to see your score, and retry to try again
- **Account management** — view your profile and sign out

## Tech Stack
- [React Native](https://reactnative.dev/) + [Expo](https://expo.dev/)
- [Expo Router](https://docs.expo.dev/router/introduction/) — file-based navigation
- [Supabase](https://supabase.com/) — auth, database (Postgres), and row-level security
